## Короткий опис

У межах практичного заняття було опрацьовано тему шаблонів і параметричного поліморфізму.

У репозиторії подано дві програми:

1. **ConsoleApplication6** — шаблонний клас для сортування одновимірного масиву за зростанням для типів `int`, `double` і `char`.
2. **ConsoleApplication7** — шаблонна функція сортування масиву методом бульбашки для різних типів даних.
